Formatações em HTML

Este documento demonstra algumas das formatações básicas que você pode usar em HTML.

Negrito e Itálico

Você pode usar negrito com a tag "strong" e itálico com a tag "em".

Texto Marcado

Para destacar texto, você pode usar a tag "mark". Por exemplo: Texto destacado.

Texto Pequeno e Grande

Para texto menor, use a tag "small". Por exemplo: Texto pequeno.

Para texto maior, use a tag "h2 ou h1". Por exemplo:

Texto Deletado e Inserido

Para texto deletado, use a tag "del". Por exemplo: Texto deletado.

Para texto inserido, use a tag "ins". Por exemplo: Texto inserido.

Texto Sobrescrito e Subscrito

Para texto sobrescrito, use a tag "sup". Por exemplo: E=mc2.

Para texto subscrito, use a tag "sub". Por exemplo: H2O.

Texto de Código

Para destacar trechos de código, use a tag "code". Por exemplo: console.log("Olá, mundo!");
